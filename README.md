# xdebug trace tk

A quick Xdebug trace viewer in python/tk

## Basic Installation & Usage

```
shell$> git clone https://github.com/emcconville/xdebug-trace-tk.git
shell$> cd xdebug-trace-tk
shell$> python xdtrace-tk.py
```
